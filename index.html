Introduction to validationJS

validationJS is a small library developed to validate text type input controls in html forms. It validates the following:

  1. URL
  2. Number
  3. E-mail
  4. U.S. Phone numbers
  5. U.S. Zip Codes
  6. Credit Cards
The library is developed purely in JavaScript and its usage is simple. It makes the use of classes to use validations.

How to use validationJS

Before starting with the usage, you need to include the library in your HTML page.

<!DOCTYPE html>
<html lang="en">
<head>
<title>validation demo</title>
</head>
<body>
<!--your HTML code should be placed here-->
<script src="js/validation.js"></script>
</body>
</html>
Remember to always include the library source at the end of body otherwise it won't run. This is to ensure that script is loaded after the complete loading of the document.

It is quite easy to use validationJS. There are different classes available for different validation types.

Class Description
validate-url It will validate the url
validate-number It will validate any numeric input
validate-email It will validate the email address
validate-phone It will validate the U.S. phone numbers
validate-zipcode It will validate the U.S. Zip code
validate-creditcard It will validate the Visa, MasterCard, American Express, Diners Club, Discover, and JCB credit cards

To make the form check these validations in the form text controls before submitting the data, you need to give a class="validate-form" to the form. There can be more than one form in the html page.

You may also use validation without forms using the library validation classes and code it according to your wish by just checking the value of the attribute error_flag in the javascript.

Using validationJS with forms

Firstly create a form in html with class="validate-form"

<form method="" action="" class="validate-form">
Then when you want to create the validations just simply use the validation classes.
<label for="text_url">Enter the URL for youtube</label> <input type="text" class="validate-url" id="text_url"/>
The above code will make the textbox active for url validation. When you will enter the data and the focus will be blurred, the data will automatically be validated and will indicate the validation status via the border of the textbox. The red border will indicate invalid data and green one will indicate valid data.

Finally, create the submit button to complete the form.

<input type="submit" value="submit"/>
The submit button will be used to submit the form data. Submitting the data is not possible until the data in the form is valid. As soon as you will try to submit the invalid data, an alert message will be displayed indicating the number of invalid data fields in the form.

Using validationJS without forms

For using validationJS without forms, you just need to make a control without a form tag and give it an appropriate validation class.

<label for="text_email">Enter your E-mail</label> <input type="text" class="validate-email" id="text_email"/> <button type="button" onclick="check_value()">check</button>
This control will also change its border to red if invalid data is entered and green if valid data is entered. You may get the element in javascript using the Id you gave to it and check its error_flag attribute. The value of the attribute is "1" if invalid data is entered and it is "0" if valid data is entered. The appropriate message can be displayed by checking this value in Javascript.

<script>
function check_value(){
var mail = document.getElementById("text_email");
if(mail.getAttribute("error_flag") === "1"){alert("Invalid E-mail entered!!!");}
else{alert("E-mail address is in correct format!!!");}
}
</script>
